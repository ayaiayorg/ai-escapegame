# AI Escape Room - AI Playground Day at SWICA

## Spiel starten

1. Oeffne die Datei **index.html** direkt im Browser (Doppelklick oder Rechtsklick > Oeffnen mit > Chrome/Edge/Firefox)
2. Teamname eingeben und auf **Abenteuer starten** klicken
3. Los gehts! Abi begleitet euch durch die 4 Stationen.

## Spielregeln

- 4 Stationen, nacheinander freischaltbar
- 25 Punkte pro Station (max. 100 + Bonuspunkte)
- Hinweise kosten je 5 Punkte
- Timer laeuft ab Spielstart
- Nach jeder Station gibt es eine lustige Mini-Challenge

## Loesungen (nur fuer Moderation!)

| Station | Loesung |
|---------|---------|
| 1 - Die geheimnisvolle E-Mail | PROMPT |
| 2 - Das chaotische Dokument | KAFFEE |
| 3 - Die Prompt-Challenge | SCHLUESSEL |
| 4 - Die Daten-Challenge | 2035 |

## Moderationsmodus oeffnen

- **Option A:** Auf dem Startscreen das kleine Feld unten anklicken, "admin" eingeben + Enter
- **Option B:** Jederzeit die Tastenkombination **Ctrl+Shift+A** druecken

Im Moderationsmodus koennt ihr:
- Alle Loesungen und Hinweise sehen
- Bonuspunkte vergeben (+5 pro Kategorie)
- Das Spiel zuruecksetzen

## Dateien

- index.html - HTML-Struktur mit Screens
- style.css - Comic-Stil Design mit SWICA-Farben
- script.js - Spiellogik, Raetsel, Abi-Kommentare

## Anpassungen

Alles Wichtige liegt uebersichtlich in script.js:
- **STATIONS** Array: Raetsel, Loesungen, Hinweise, Punkte
- **ABI_COMMENTS**: Alle Abi-Sprueche
- **ERROR_MESSAGES**: Lustige Fehlermeldungen
- **FUN_TASKS**: Mini-Challenges nach jeder Station
